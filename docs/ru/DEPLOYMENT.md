## Назначение

Этот документ описывает, как развернуть Family Health OS с нуля.

Цель — получить рабочую систему из личных Google Sheets, одной семейной Google Sheets и ChatGPT-проекта, который работает по спецификациям из GitHub.

Репозиторий проекта:

```text
https://github.com/MaximKulmars/Family-Health-Os
```

---

## Что нужно до начала

Потребуется:

- аккаунт Google;
- доступ к Google Sheets;
- ChatGPT-проект;
- доступ к репозиторию проекта;
- список участников семейной системы.

---

## Шаг 1. Создать личные таблицы

Создайте по одной Google Sheets на каждого участника.

Рекомендуемая схема именования:

```text
<Name>_health
```

Пример:

```text
User_health
Partner_health
Child_health
```

Личная таблица используется для подробных данных одного человека.

---

## Шаг 2. Создать семейную таблицу

Создайте отдельную Google Sheets:

```text
Family_health
```

Эта таблица используется только для координации: общие задачи, статусы, ближайшие проверки, семейные решения, общие ресурсы и ссылки на личные таблицы.

Подробные личные данные в семейной таблице не хранятся.

---

## Шаг 3. Создать вкладки в личных таблицах

В каждой личной таблице создайте одинаковые вкладки:

```text
Dashboard
Members
Measurements
Laboratory
Nutrition
Activity
Sleep
Symptoms
Medications
Supplements
Medical Events
Decisions
Goals
Attachments
Settings
Baseline Report
Current Strategy
Weekly Review
Ideas & Questions
```

---

## Шаг 4. Создать вкладки в семейной таблице

В `Family_health` создайте вкладки:

```text
Dashboard
Family Members
Open Tasks
Upcoming Checks
Recent Signals
Family Decisions
Medical Visits
Shared Resources
Attachments
Settings
```

---

## Шаг 5. Настроить идентификаторы

Используйте стабильные `member_id`.

Пример:

```text
user
partner
child
family
```

Один человек — один стабильный идентификатор во всех таблицах.

Для детского профиля добавьте в настройки таблицы пометку:

```text
child_profile | true
```

---

## Шаг 6. Настроить ChatGPT-проект

В инструкции ChatGPT-проекта вставьте содержимое файла:

```text
docs/MASTER_PROMPT.md
```

В инструкции должна быть ссылка на репозиторий проекта.

---

## Шаг 7. Проверить запуск

Отправьте тестовое сообщение:

```text
Я: Пользователь
Вес сегодня 93 кг
```

Ассистент должен определить:

1. Кому принадлежат данные
2. Какая личная таблица нужна
3. Какая вкладка нужна
4. Нужно ли обновить аналитику или семейные задачи

---

## Принцип приватности

GitHub содержит систему и документацию.

Google Sheets содержат рабочие данные.

Персональные и медицинские данные нельзя сохранять в GitHub.
